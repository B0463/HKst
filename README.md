# HKst

## configs

* create `\ssl\` directory with your domain ssl

## node modules

* typescript

* express

* express cookie-parser

* fs

* http

* https

## types

* @types/node

* @types/express

* @types/cookie-parser

* @types/http

* @types/https

## run

* npm install `<node modules>`

* npm install `<types>`

* tsc --init `save tsconfig.json berofe`

* cd ./server_dist/

* node server.js
