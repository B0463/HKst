validação de cpf

O CPF consistem em 11 digitos, que é 9 digitos principais e 2 validadores conhecido como numero validador.

Para verificar se um CPF é realmente verdadeiro, é preciso fazer um calco nos 9 digitos para encontrar o verificador referente.

Para validar o primeiro digito verificador encontrado em X de:

123.456.789-X0

É preciso fazer a multiplicação dos numeros por 10, 9, 8, 7... da seguinte forma:

1x10=10 2x9=18 3x8=24 4x7=28 5x6=30 6x5=30 7x4=28 8x3=24 9x2=18

Apos isso é preciso fazer a soma do resultado de todas as multiplicações:

10+18+24+28+30+30+28+24+18=207

Apos isso é preciso dividir o resultado no caso "207" por 11 mas pegar o resto da divizão

207%11=9

E por fim vamos pegar 9 e retirar de 11

11-9=2

Logo o primeiro digito verificador é 2.

Para pegar o segundo digito verificador é preciso multiplicar os primeiros 10 digitos do CPF por 11, 10, 9, 8...

1x11=11 2x10=20 3x9=27 4x8=32 5x7=35 6x6=36 7x5=35 8x4=32 9x3=27 0x2=0

Apos isso efetuar a soma de todos os resultados

11+20+27+32+35+36+35+32+27+0=255

Agora pegar oresto da divisão de "255" por 11

255%11=2

Aagora subtrair o resultado de 11

11-2=9

No caso o resultado é 9. caso o resultado od primeiro ou segundo digito de um valor igual ou maior que 10 ele vira 0. nesse caso o segundo digito é "9"

Sendo assim os numeros verificadores do cpf 123.456.789-00 é 29 e nesse caso como o do cpf é 00 e o resultado é 29.o cepf é invalido.

cpf 111.111.111-11 ou 222.222.222-22 ou 333.333.333-33 eadiante são validos mas não estão registrados oficialmente.

validando com JS

function TestaCPF(strCPF) {
let soma;
let resto;
soma = 0;
if(strCPF == "00000000000"}) {
return false
};
for(let i=1;i<=9;i++) {
soma = soma + parseInt(strCPF.substring(i-1, i))*(11-i);
}
resto = (soma*10)%11;
if((resto == 10)||(resto == 11)) {
resto = 0;
}
if(resto != parseInt(strCPF.substring(9, 10))) {
return false;
}
soma = 0;
for(let i=1;i<=10;i++) {
soma = soma + parseInt(strCPF>.substring(i-1, i))*(12-i);
}
resto = (soma*10)%11;
if((resto == 10)||(resto == 11)) {
resto = 0;
}
if(resto != parseInt(strCPF.substring(10, 11))) {
return false;
}
return true;
}

No codigo acima tem a função "TestaCPF" que tem o parametro "strCPF" que é uma string.

Caso o CPF seja valido ela devolve true, caso contrario ela devolve false.
